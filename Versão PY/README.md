# Versão PY — Módulo Python do Sistema DAC

Esta pasta contém a implementação em Python do Sistema DAC.

Estrutura:
- `src/`: código-fonte (database, modules, ui, utils)
- `tests/`: testes (unit, integration, performance)
- `data/`: dados e banco local
- `recursos/`: arquivos auxiliares
- `web/backend/`: serviços backend auxiliares

Observação: o ambiente virtual `.venv/` e dependências não são versionados. Instale com `pip install
